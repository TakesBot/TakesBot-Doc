# /bind

此命令用于绑定至舞萌DX账号

1.进入 `舞萌 | 中二` 公众号

2.点击`获取二维码`

3.长按 -> 识别图中的二维码

获取到以 `SGWCMAID` 为开头的字符串，长按复制

4.使用命令 `/bind 刚才获取的字符串` 绑定

![image-20251021013606831](https://cn-sy1.rains3.com/takesbot/2025/10/b11f0c4a257acce77c20366d3f56e21d.png)

注：如果你正在群聊中使用这个，请**一定**要撤回你发送的内容
