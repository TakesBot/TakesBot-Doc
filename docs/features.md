# 功能介绍
## 绑定

使用  `/bind` 来绑定舞萌账号

`/bindlx` 绑定落雪查分器

`/bindsy` 绑定水鱼查分器

当绑定完成后，你可以使用 `更换数据源` 命令来更换不同的数据源

如：`更换数据源水鱼`

## Best50

`/b50` 可以获取一张玩家最佳50首乐曲的图片

（此功能将会扣取积分，更换数据源在此处适用）

`/pc50` 可以获取游玩次数最多的50首乐曲

（此功能要求绑定舞萌账号，且该功能会扣取积分）
